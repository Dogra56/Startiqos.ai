# StartIQOS.ai — Early-Stage Founder OS

## Project Structure

```
startiqos/
├── index.html                  ← Main landing page
├── pages/
│   ├── privacy.html            ← Privacy policy
│   └── terms.html              ← Terms of use
└── assets/
    ├── css/
    │   ├── tokens.css          ← Design tokens & CSS variables
    │   ├── base.css            ← Reset & global styles
    │   ├── animations.css      ← Keyframes & scroll reveal
    │   ├── components.css      ← Reusable UI components
    │   └── layout.css          ← Section & page layouts
    └── js/
        ├── blob.js             ← Animated canvas background engine
        └── main.js             ← Core interactions & live demos
```

## How to Run

1. Open the `startiqos/` folder in VS Code
2. Install the **Live Server** extension (ritwickdey.LiveServer)
3. Right-click `index.html` → **Open with Live Server**
4. Site opens at `http://127.0.0.1:5500/index.html`

> No build step required. Pure HTML/CSS/JS.
